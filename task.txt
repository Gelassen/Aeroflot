Список используемых технологий:
Python 3.13
FastAPI
Apache Kafka
PostgreSQL 13+
Docker

Задание:
Создать набор docker-контейнеров, содержащих следующие сервисы:
1. [done] Очередь сообщений, которые принимает следующий тип сообщения - Событие Inventory (время, номер рейса, дата вылета, класс бронирования, количество свободных мест)
2. [done] Контейнер, в котором запущен инстанс PostgreSQL, с таблицей для сохранения данных Inventory
3. [done] Сервис, слушающий очередь и обновляющий информацию в базе о свободных местах для заданного рейса и класса бронирования.
4. [done] Сервис, предоставляющий доступ к данным БД на чтение в виде REST API, реализованном с использованием FastAPI
